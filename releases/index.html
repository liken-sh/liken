liken releases

This is the release channel for liken, a small operating system that boots a machine straight into Kubernetes. The channel is plain object storage: every published file is public, and a release never changes after it is published.

The layout:

  • channel.yaml names the newest published version.
  • /<version>/release.yaml names a release's artifacts by sha256 digest, and the artifacts sit beside it.
  • /<version>/LICENSES.md carries the third-party notices for everything a release redistributes.
  • /sources/<component>/<version>/ serves the corresponding source for the GPL- and LGPL-licensed components, beside the binaries built from it.

The repository explains how it all works, and GETTING-STARTED.md walks the path from a release to a running cluster.
