liken

An operating system that boots straight into Kubernetes, written to be read.

The whole system — a Go init, a vendored kernel, and the build that assembles them into a bootable image — lives in one repository, commented generously enough to teach how a Linux system boots and how Kubernetes takes over from there.

Read it on GitHub. The server you are reading this from is a liken cluster, built from that repository.

By and by this site will grow the rest of the story: released images to download, the documentation arranged for reading, and a path from nothing to a running cluster of your own.
