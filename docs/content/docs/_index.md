# The `liken` manual

This manual tells you how to set up and run a `liken` cluster. The
guides give the steps for the common tasks. The reference describes
the API resources, the release channel, and the `liken` command.

Read [Install a cluster](/docs/guides/install/) first. It takes you
from a downloaded release to `kubectl get nodes`.

The manual is deliberately small. The repository is the deep
documentation: every script, manifest, and program in
[liken-sh/liken](https://github.com/liken-sh/liken) carries comments
that explain how the system works. Where this manual answers "how",
the repository answers "why".

Every page of this site is also served as Markdown. Append `index.md`
to a page's address to get it. [/llms.txt](/llms.txt) lists those
pages, and [/llms-full.txt](/llms-full.txt) holds the whole manual in
one file.
